### 一、Web Worker 介绍

Web Worker 使得一个 Web 应用程序可以在与主执行纯种分享的后台线程中运行一个脚本操作。

### 二、Web Worker 分类

Web Worker 一共包含了 3 种类型的 Worker：

- Delicated Worker（专用 Worker）
- Shared Worker（共享 Worker）
- Service Worker（服务 Worker）
