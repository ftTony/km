#### 1.1 流

**“流”又叫文档流，是 css 的一种基本定位和布局机制。**流是 html 的一种抽象概念，暗喻这种排列布局方式好像水流一样自然自动。“流体布局”是 html 默认的布局机制，如你写的 html 不用 css，默认自上而下（块级元素如`div`）从左到右（内联元素如`span`）堆砌的布局方式。
