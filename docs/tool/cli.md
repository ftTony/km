## 内容

### 第三方库

- `commander`：能够更好地组织和处理命令行的输入。
- `co-prompt`：使用这个工作可以自动提供提示信息，并且分步接收用户的输入，体验类似`npm init`时的一步一步输入参数的过程。也可以使用`inquirer`
- `chalk`：修改控制台输出内容样式
- `download-git-repo`：用来下载远程模板
